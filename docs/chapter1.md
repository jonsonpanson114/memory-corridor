# 記憶の回廊
## 第一章「館の扉」— 完全台本

> 記憶術：場所法（Memory Palace）  
> 全5話 + 隠しエピソード1話  
> 推定プレイ時間：1話あたり10〜15分

---

## 登場人物

**僕（語り手）**  
記憶を失った状態で目覚める。何かから逃げてきた気配がある。
感情は抑制されているが、ミラの言葉に少しずつ揺れていく。

**ミラ**  
黒髪、白いワンピース。館に最初から存在している。
知っていることをすべては語らない。温かいが、どこか遠い。
「僕」の記憶の深いところから来た存在——だが、本人はそれを言わない。

## 第一話「目覚め」

### 冒頭ナレーション

```
目が覚めると、僕は自分の名前を思い出せなかった。

天井は古い木材で組まれていて、どこかの隙間から夜の光が落ちている。
体を起こすと、見知らぬ部屋だとわかった。でも——妙に懐かしかった。

ベッドの脇のテーブルに、一枚の紙が置いてある。
```

### 紙の内容（画面中央に独立表示）

```
『記憶を取り戻したければ、館の奥へ進め。
 ただし——忘れるな。
 ここはお前が作った場所だ。』
```

### ナレーション続き

```
意味がわからなかった。
「僕が作った場所」という言葉だけが、頭の中で妙な重さを持った。

廊下に出ると、遠くで何かが燃えているような匂いがした。
蝋燭の匂い。燻んだ紙の匂い。

そして——誰かの気配。
```

### ミラとの出会い

```
[ナレーション]
廊下の突き当たりに、女性が立っていた。
黒い髪が背中まで伸びている。白いワンピース。
窓の外の暗闘を、ただ静かに見ていた。
```

**ミラ：**
「来ると思っていました。」

```
[ナレーション]
振り返らないまま、彼女は言った。声は低く、穏やかだった。
```

**ミラ：**
「遅かったですね。もう少し早く目が覚めると思っていた。」

```
[ナレーション]
僕は何も言えなかった。
```

### 選択肢A

> **「あなたは誰ですか」**

**ミラ：**
「……それを聞くんですね。」

```
[ナレーション]
ミラはゆっくりと振り返った。
その目は、暗闇の中でも色がわかった。深い、緑色。
```

**ミラ：**
「私の名前はミラ。それ以上のことは——今は、言えません。」

**ミラ：**
「でも、あなたが知っている人間であることは確かです。」

```
[ナレーション]
その言葉が、胸の奥で何かに触れた。
知っている——？　どこで？　いつ？
記憶の端を引っ張ると、するすると消えていく。
```

```
[ナレーション]

```

**ミラ：**

```
[ナレーション]
```

## 第二話「場所の記憶」

### 冒頭（前回の振り返り）

```
[薄暗い画面に文字だけ]
昨日、僕は五枚の絵を記憶した。
今日もまた、夢の中で館に戻ってきた。
```

### 本編ナレーション

```
廊下はまだ薄暗かった。
でも昨日より、少しだけ景色が見えた。

柱の傷。窓の外の枯れた庭。
壁紙の剥がれた模様が、何かの地図に見えた。

ミラはいつもの場所にいた。
今日は窓ではなく、古い本棚の前に立っていた。
```

**ミラ：**
「覚えていますか。昨日の絵。」

```
[ナレーション]
僕はうなずいた。
……正確には、覚えているつもりだった。
```

**ミラ：**
「ちゃんと自分の場所に置きましたか？」

**ミラ：**
「記憶というのは不思議なもので——置く場所を間違えると、二度と見つからなくなる。」

```
[ナレーション]
ミラは本棚から古い本を一冊取り出した。
表紙には何も書いていなかった。
```

**ミラ：**
「これを読むといい。場所と記憶の話が書いてある。」

### 本の内容（画面に独立表示・古い書体風）

```
『古代ギリシャの弁論家シモニデスは、宴会の途中で席を外した。
 その直後、建物が崩れ落ちた。
 遺体は判別がつかないほどに潰れていたが——
 シモニデスは、誰がどこに座っていたかを正確に思い出すことができた。
 
 場所が、顔を保存していたのだ。
 
 それ以来、記憶術者たちは「場所」を使って記憶を保存するようになった。
 宮殿でも、街でも、廊下でも——歩ける場所なら、どこでも記憶の器になる。』
```

### 選択肢B

> **「これは本当の話ですか」**

**ミラ：**
「シモニデスの話は本当です。ただ——」

**ミラ：**
「あなたにとって大事なのは、それが本当かどうかではなく、使えるかどうかでしょう。」

```
[ナレーション]
ミラは珍しく、少し笑った。
その笑い方を——僕は、どこかで見たことがある気がした。
```

---

> **「なぜあなたはこれを知っているんですか」**

**ミラ：**
「……。」

```
[ナレーション]
ミラは少し間を置いた。
答えを探しているのか、答えを選んでいるのか、わからなかった。
```

**ミラ：**
「あなたが、知っていたからです。」

```
[ナレーション]
その言葉の意味を聞こうとした瞬間、ミラは話題を変えた。
```

---

### 共通ルートへ

**ミラ：**
「今日は、もう少し難しいことをお願いします。」

**ミラ：**
「昨日より多く。そして——細かいところまで。」

```
[ナレーション]
ミラは本棚の後ろから、小さな箱を取り出した。
中には紙片が入っていた。文字が書いてある。
```

**ミラ：**
「これを、あなたの場所に置いてください。
 今日はただ置くだけじゃなく——なぜそこに置くのかを、自分に問いかけながら。」

```
[ナレーション]
「なぜ置くのか」——そんなことを考えながら記憶するのか。
でも、ミラの言うことには、いつも理由がある気がした。
```

### 記憶術フェーズへの橋渡し

```
その夜、僕は目を閉じた。
自分の場所——僕がよく知っている場所——を思い浮かべながら。

ミラは言っていた。「奇妙なシーンほど残る」と。
だから今日は、大げさに想像してみた。
本当に馬鹿馬鹿しいくらいに。

……不思議なことに、そちらの方が映像が鮮やかだった。
```

### 第二話エンディング

```
翌朝、夢の中で確認した。
いくつかは残っていた。
いくつかは——もう、霧の中にあった。

「場所の精度が、記憶の精度だ」とミラは言っていた。
その意味が、少しだけわかった気がした。
```

### 次回予告
> 「明日——ミラは、僕の名前を知っているような言い方をした。」

## 第三話「声の断片」

### 冒頭

```
三日目の夢は、少し違った。

館が、前より広かった。
いや——広くなったのではなく、見えていなかっただけかもしれない。

廊下の奥に、新しい扉があった。
昨日まで気づかなかった。あるいは、なかったのか。
```

### ミラとの対話

**ミラ：**
「気づきましたね、新しい扉に。」

```
[ナレーション]
今日のミラは、扉のそばに立っていた。
手を後ろで組んで、何かを待っているように。
```

**ミラ：**
「記憶が戻ると、見えるものが増えます。
 館は——あなたの記憶そのものですから。」

```
[ナレーション]
僕はその言葉を黙って聞いた。
「館はあなたの記憶そのもの」——それが本当なら。
この館が広くなるということは、僕の記憶が戻っているということだ。
```

**ミラ：**
「扉の先には行けません。まだ。」

**ミラ：**
「でも——今日は、音が聞こえるかもしれない。」

```
[ナレーション]
ミラは扉に耳を当てた。
僕も近づいた。
扉の向こうから、かすかに声がした。
誰かの声。知っているような気がする声。
でも、言葉になっていなかった。
```

### 選択肢C

> **「あの声は誰ですか」**

**ミラ：**
「……聞こえましたか。」

```
[ナレーション]
ミラの表情が、わずかに変わった。
驚きと——何かほかのものが、混ざっていた。
```

**ミラ：**
「まだ、答えるべき時ではない。でも——」

**ミラ：**
「その声が聞こえたということは、あなたの記憶が、正しい方向に向いています。」

---

> **「もっとよく聞こうとしたら、消えた」**

**ミラ：**
「そうでしょう。」

**ミラ：**
「記憶は、追いかけると逃げます。
 静かに待っていると——向こうからやってくる。」

```
[ナレーション]
ミラはそう言って、扉から離れた。
その言葉は、記憶術の話のようでもあり、別の何かの話のようでもあった。
```

---

### 共通ルートへ

**ミラ：**
「今日の課題は、少し違います。」

**ミラ：**
「これまでは、ものを覚えてもらいました。
 今日は——順番を覚えてもらいます。」

```
[ナレーション]
ミラはポケットから七枚の紙片を取り出した。
それぞれに何かが書いてある。
```

**ミラ：**
「順番が一つでも違うと、扉は開きません。
 記憶は、細部が正確でなければ、鍵にならない。」

```
[ナレーション]
七枚——昨日より二枚多い。
でも不思議と、怖くなかった。
自分の場所に、七つの地点を思い浮かべた。
それぞれに、ひとつずつ置いていけばいい。
```

### 第三話エンディング

```
目を覚ます前に、扉の向こうの声をもう一度聞こうとした。
でも——ミラの言った通り、追いかけると消えた。

代わりに、一つだけ映像が浮かんだ。
台所の窓。朝の光。誰かの手が、カップを持っている。

誰の手だろう。
```

### 解放される記憶の断片
> *台所の朝の光。誰かがそこにいた。*

## 第四話「忘れることの意味」

> ※ この話は感情的なクライマックスに向かう転換点。  
> 選択肢の答えによって、第五話のミラの態度が変わる。

### 冒頭

```
四日目。

起きると、昨日の声がまだ耳に残っていた。
夢と現実の境界が、少しずつ曖昧になってきた。
それが怖いのか、心地いいのか、自分でもわからなかった。

館はまた広くなっていた。
廊下の左に、新しい部屋があった。
扉は半開きになっていて、中から光が漏れていた。
```

### ミラとの対話

**ミラ：**
「今日は、聞いてもいいですか。」

```
[ナレーション]
これまでと違う入り方だった。
ミラが何かを聞いてくることは、なかった。
```

**ミラ：**
「あなたは——本当に、思い出したいんですか。」

```
[ナレーション]
その言葉が、想像以上に重かった。
「思い出したい」と思っていた。当然のように。
でも——ミラにそう聞かれると、本当にそうなのか、急にわからなくなった。
```

### 選択肢D（この選択が第五話のミラのトーンに影響する）

> **「思い出したい。だから来ている。」（積極的）**

**ミラ：**
「……そうですか。」

```
[ナレーション]
ミラは少し間を置いた。
何かを確認するように、僕の目を見た。
```

**ミラ：**
「なら——教えておかなければならないことがあります。」

**ミラ：**
「思い出すということは、忘れたかったことも、戻ってくるということです。
 それでも、ですか。」

→ **[フラグ：積極的]** ミラが第五話でより深い真実を語る

---

> **「わからない。でも、ここにいる。」（迷い）**

**ミラ：**
「正直ですね。」

**ミラ：**
「わからないまま進むのは、勇気がいることです。確信があるより、ずっと。」

```
[ナレーション]
ミラは部屋の方に目をやった。
その横顔に、初めて何か柔らかいものが見えた気がした。
```

**ミラ：**
「……私も、最初はわからなかった。」

```
[ナレーション]
「最初」——ミラにも「最初」があったのか。
聞こうとしたが、ミラはもう話を変えていた。
```

→ **[フラグ：迷い]** ミラが第五話で自分自身の秘密を少しだけ明かす

---

**ミラ：**
「今日の課題は——記憶の宮殿を、広げることです。」

```
[ナレーション]
```

**ミラ：**

```
[ナレーション]

```

### 第四話エンディング

```
書斎の本棚に、一冊だけ開いた本があった。
```

## 第五話「扉の向こう」（第一章クライマックス）

> ※ 第四話の選択によって、ミラのセリフが変化する分岐あり。  
> 記憶術トレーニングは第一章の集大成（難易度：高）。

### 冒頭

```
五日目。

館が、また広くなっていた。
今日は廊下全体が見えた。
最初の夜、暗くてわからなかった場所に、
いくつもの扉があることがわかった。

ミラは廊下の中央に立っていた。
いつもと違う場所に。
```

### ミラとの対話（積極フラグ）

**ミラ：**
「今日で、第一章が終わります。」

```
[ナレーション]
「章」という言葉を、ミラは自然に使った。
まるでこれが物語であることを、知っているかのように。
```

**ミラ：**
「あなたが積極的に思い出そうとしているのは、わかっています。
 だから——少しだけ、教えます。」

**ミラ：**
「扉の向こうには、あなたが最も強く覚えている場所があります。
 そしてその場所に、あなたが最も避けてきた記憶がある。」

```
[ナレーション]
「避けてきた記憶」——その言葉が、胸の奥で何かに触れた。
怖かった。でも——知りたかった。
```

---

### ミラとの対話（迷いフラグ）

**ミラ：**
「今日で、第一章が終わります。」

**ミラ：**
「……一つだけ、言っていいですか。」

```
[ナレーション]
ミラの声が、少しだけ変わった。
いつもより低く。いつもより近かった。
```

**ミラ：**
「私は——あなたが作った存在です。」

```
[ナレーション]
その言葉の意味を、すぐには理解できなかった。
```

**ミラ：**
「あなたが記憶の中に作り上げた、もう一人の自分。
 だから私は、あなたの記憶をすべて知っている。」

**ミラ：**
「でも——知っているだけでは、どうにもできない。
 あなた自身が、思い出さなければならない。」

```
[ナレーション]
涙が出そうになった理由が、少しわかった。
ミラは——僕自身だったのか。
```

---

### 共通ルートへ

**ミラ：**
「最後の課題です。これまでで一番難しい。」

**ミラ：**
「でも——あなたは、もう場所法を知っている。
 自分の宮殿を持っている。
 あとは、信じるだけです。」

```
[ナレーション]
ミラは廊下の突き当たりの扉を指した。
さっきまで気づかなかった扉。
鍵穴から、光が漏れていた。
```

**ミラ：**
「これを開けるには、今日の課題をクリアしなければなりません。
 記憶が鍵になる。」

**ミラ：**
「行ってきてください。私はここで待っています。」

### 記憶術フェーズ（第一章集大成）

```
[ナレーション]
廊下を歩きながら、これまでのことを思い出した。

そのことが、今は確かにわかった。
```
