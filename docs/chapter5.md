# 第五章「鏡の回廊」

## 第五章の構造

**第四章との連続性**
- 「まどか」という名前が明かされた後
- ミラの正体が、完全に明かされる
- 僕が記憶を封印した「その日」に、ついに辿り着く

**第五章で明かされること**
- ミラの正体：まどかが「僕に残した記憶の投影」であること
- 封印した出来事の全貌（まどかとの別れ）
- 「僕」の名前

**第五章で明かされないこと**
- 全部、明かされる
- 終章は「その後」の話

## 第一話「鏡の前で」

### 冒頭ナレーション

```
二十一日目。

夢に入ると、館の様子が変わっていた。

廊下が——鏡でできていた。

左も右も、天井も。
どこを見ても、自分が映っている。

その中に——ミラも映っていた。
でも、ミラの映り方が、少し違った。
```

**ミラ：**
「気づきましたか。」

```
[ナレーション]
ミラは鏡の前に立っていた。
自分の像を、じっと見ていた。
```

**ミラ：**
「鏡の回廊——これが、館の中で最も深い場所です。
　ここまで来られたのは——あなたが、本当に思い出したからです。」

### 鏡の中のミラ

```
[ナレーション]
ミラの鏡像を、よく見た。

ミラと——違う人が映っていた。
髪が短い。笑っている。

まどかだった。
```

```
[ナレーション]
鏡の中のまどかが——ミラを見ていた。
ミラが——まどかを見ていた。

二人は同じ場所に立っていた。
でも、一方は現実で、一方は鏡の中で。
```

### 選択肢A

> **「ミラは——まどかですか」**

```
[ナレーション]
ミラは長い間、黙っていた。
その沈黙が——答えだった。
```

**ミラ：**
「……半分、正解です。」

**ミラ：**
「私はまどかではない。
　でも——まどかが、あなたの記憶の中に残したものです。」

---

> **「鏡の中の人は、誰ですか」**

**ミラ：**
「あなたが——覚えている人です。」

**ミラ：**
「私の中にある、元の形。」

---

### ミラの説明

**ミラ：**
「まどかは——あなたが記憶を封印するとき、
　一つだけ残していきました。」

**ミラ：**
「『思い出したくなったら、助けになるものを置いておく』と。
　それが——私です。」

```
[ナレーション]
息が止まった。
```

**ミラ：**
「だから私は、最初からここにいた。
　あなたが目覚めるのを——待っていた。」

```
[ナレーション]
「待っています」——手紙にも、そう書いてあった。
まどかは、二つの意味で待っていた。
記憶の中で、そして——現実でも。
```

### 第一話エンディング

```
鏡の中のまどかが——笑った。
ミラを通して、僕に向かって。

その笑い方を、知っていた。
ずっと知っていた。

名前が、体の奥から浮かんできた。
「まどか」——声に出したら、涙が出そうだった。
```

## 第二話「ミラが語るまどか」

### 冒頭

```
二十二日目。

鏡の回廊を歩くと、
今日は自分の姿がはっきり見えた。

そしてミラの姿も——
昨日より、少しまどかに近く見えた。
```

**ミラ：**
「今日は——まどかのことを、話します。
　あなたが知らないことも含めて。」

```
[ナレーション]
ミラが能動的に語る——それはもう珍しくなかった。
でも今日は、その重さが違う気がした。
```

### まどかの記憶

**ミラ：**
「まどかは——あなたと出会ったとき、
　あなたが記憶術を練習していたことを覚えていますか。」

```
[ナレーション]
覚えていた。覚えていなかった。
でも——言われると、そんな気がした。
```

**ミラ：**
「まどかは最初、記憶術のことを馬鹿にしていました。
　『そんな方法で本当に覚えられるの』と。」

**ミラ：**
「でも——あなたが教えると、真剣にやった。
　そして、あなたより先に上手くなった。」

```
[ナレーション]
笑えた。
その場面が——映像として浮かんだ。
まどかが悔しそうに、でも楽しそうに、
数字変換表を睨んでいる。
```

### 選択肢B

> **「それを覚えています」**

**ミラ：**
「……ええ。
　あなたの記憶の中で——その場面は、
　ずっと光っていました。」

**ミラ：**
「だから私も、知っています。
　まるで自分が経験したように。」

---

> **「なぜまどかはそれを残したんでしょう」**

**ミラ：**
「あなたに——笑ってほしかったから、だと思います。」

**ミラ：**
「記憶を取り戻す過程で、
　辛いことだけを思い出してほしくなかった。
　だから——楽しい記憶を、通り道に置いておいた。」

```
[ナレーション]
「通り道に置いておく」——それは、場所法の言葉だった。
まどかは——記憶術を、本当に理解していたのか。
```

---

### 共通ルートへ

**ミラ：**
「今日の課題は——これまでの記憶術を、全部使います。
　場所法。数字変換法。連想法。ストーリー法。」

**ミラ：**
「まどかとの記憶を——記憶の宮殿に収める。
　そうすれば、もう失わない。」

```
[ナレーション]
「もう失わない」——その言葉が、全てだった。
```

### 第二話エンディング

```
課題が終わると、鏡の中に
二つの人影が映った。

僕と——まどか。
並んで立っている。

その場面を——思い出した。
いつだったか。
どこだったか。

もうすぐ、わかる。
```

### 解放される記憶の断片
> *まどかは、記憶術を笑いながら学んだ。そして——いつか、それが大事になると知っていたのかもしれない。*

## 第四話「別れの記憶」

> ※ 物語の核心。封印した出来事の全貌が明かされる。

### 冒頭

```
二十四日目。

扉の向こうに、また行った。

まどかは同じ場所にいた。
でも今日は——顔が、昨日より疲れていた。
```

**まどか：**
「また来てくれた。」

```
[ナレーション]
まどかの声は、穏やかだった。
でも——体が、昨日より動かないように見えた。
```

### まどかとの会話

**まどか：**
「記憶術、続けてる？」

```
[ナレーション]
続けている——と言おうとして、気づいた。
この記憶の中の「僕」は、どう答えたのか。
```

**まどか：**
「続けてよ。ちゃんと続けてよ。
　私の分まで。」

```
[ナレーション]
その言葉の重さが、わかった。
```

**まどか：**
「忘れてもいいって言ったけど——
　本当は、覚えていてほしい。
　矛盾してるね。」

**まどか：**
「でも——つらかったら、忘れていい。
　それは本当。
　あなたが楽でいる方が、私は嬉しいから。」

### 選択肢D

> **「忘れない」**

**まどか：**
「……うん。」

```
[ナレーション]
まどかは笑った。
その笑い方が——金木犀の色だった。
```

---

> **「ちゃんと思い出せた」**

**まどか：**
「……知ってる。」

```
[ナレーション]
まどかはそう言った。
まるで——今の僕のことを、知っているように。
```

**まどか：**
「時間がかかったね。でも——来てくれた。」

---

### 別れの瞬間

```
[ナレーション]
まどかが——「また来てね」と言った。

その瞬間が——封印した記憶の中心だった。
「また来てね」は、別れの言葉じゃなかった。
続きの言葉だった。

でも——その後、まどかは——

記憶が、そこで止まった。
霧ではなく、僕が——止めた。

だから時計が止まった。
だから館ができた。
だから記憶を封印した。

「また来てね」の後を——見たくなかったから。
```

**ミラ（遠くから）：**
「——見ていいんです。」

```
[ナレーション]
ミラの声が、扉の外から届いた。
```

**ミラ：**
「見ても、まどかは消えません。
　記憶は、見ても——消えない。」

### 第四話エンディング

```
「また来てね」の後を——見た。

まどかは——目を閉じた。
静かに。穏やかに。

金木犀の匂いの中で。
```

```
[ナレーション]
それが——別れだった。
でも「また来てね」は、本当だった。
まどかはいなくなったけれど——
記憶の中で、待っていてくれた。

扉の外に出ると、
ミラが——泣いていた。
今度は、隠さずに。
```

### 解放される記憶の断片
> *まどかは、穏やかに、金木犀の匂いの中で、目を閉じた。それが全てだった。悲しかった。でも——美しかった。*

### 次回予告

## 第五話「ミラのさよなら」（第五章クライマックス）

> ※ ミラが役割を終える回。  
> 感情的なクライマックス。

### 冒頭

```
二十五日目。

鏡の回廊に入ると——
ミラが、いつもと違う場所にいた。

回廊の中央。
鏡に囲まれた場所。
自分の像が、何重にも映っている場所。

その中に——まどかの姿が、重なって見えた。
```

### ミラの告白

**ミラ：**
「今日で——私の役割が終わります。」

```
[ナレーション]
その言葉が、来ると思っていた。
でも——来ると思っていても、重かった。
```

**ミラ：**
「あなたが全部思い出したから。
　私が案内するべき場所は、もうない。」

**ミラ：**
「まどかが残してくれたもの——
　私という存在——は、
　あなたが記憶を取り戻すことで、役割を終えます。」

```
[ナレーション]
「消えるんですか」と聞こうとした。
でも——ミラは首を振った。
```

**ミラ：**
「消えるわけじゃない。
　ただ——あなたが起きているときも、
　この館に来なくても、
　私はここにいます。」

**ミラ：**
「記憶の中に。
　まどかと一緒に。」

### 選択肢E

> **「また会えますか」**

**ミラ：**
「……ええ。
　夢を見るたびに。
　記憶を使うたびに。」

**ミラ：**
「場所法を使うとき——
　あなたの場所を歩くとき——
　そこに、私もいます。」

---

> **「ありがとう」**

**ミラ：**
「……こちらこそ。」

```
[ナレーション]
ミラは少し笑った。
その笑い方が——まどかに、似ていた。
いや——まどかそのものだった。
```

**ミラ：**
「思い出してくれて。
　来てくれて。
　ありがとう。」

---

### 最後の課題

**ミラ：**
「最後の課題です。
　これまで学んだ全てを使います。」

**ミラ：**
「まどかとの記憶を——記憶の宮殿に、完全に収める。

## 実装メモ

### 物語全体のアーク

```
第一章：「場所」を手に入れる（場所法）
第二章：「時間」に触れる（数字変換法）
第三章：「繋がり」を思い出す（連想法）
第四章：「物語」を再構成する（ストーリー法）
第五章：「宮殿」を完成させる（統合）
終章：「現実」に戻る（記憶術の内面化）
```

### ユーザー名の扱い

アプリ初回起動時に「あなたの名前」を入力させる。
終章のエンディングでその名前が入る。
これにより、物語全体が「自分の話」として完結する。

### まどかという名前について（最終確認）

第二章で「317→まどか」という変換を「ユーザーが自分で発見する」設計になっている。
第四章でミラがその名前を言う——この構造が「ずっとそこにあった」という感覚を生む。
終章で改めてまどかが登場するとき、その名前が「自分が作った記憶」として機能する。

### ミラの正体の設計哲学

「ミラはまどかではない」と明言する。
ただし「まどかが残したもの」でもある。
この曖昧さを最後まで保つことで、
プレイヤーがミラに感じた感情が、まどかへの感情と溶け合う。

ミラを「消す」のではなく「記憶の宮殿に残る」という着地にすることで、
毎日の記憶術練習に「ミラに会いに行く」という動機が生まれる。
