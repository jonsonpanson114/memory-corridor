## 第二章の構造

**第一章との連続性**
- 台所にいた「誰か」の正体に、一歩近づく
- ミラが「その日付」を知っていることが明らかになる
- 館の新しいエリア「時計塔」が解放される

**第二章で明かされること**
- 止まった時計が示す日付：ある秋の午後
- その日付に「僕」がいた場所
- ミラが初めて、自分の感情を漏らす瞬間

**第二章で明かされないこと**
- 台所にいた人物の顔
- なぜ時計が止まったのか
- ミラが「僕が作った存在」であることの詳細

### 数字変換表（画面に独立表示）

```
数字変換法（Major System）の基本

0 = s, z の音　（ゼロのz）
1 = t, d の音　（縦棒が1本）
2 = n の音　　（縦棒が2本）
3 = m の音　　（縦棒が3本）
4 = r の音　　（fourのr）
5 = l の音　　（手のひらを広げると5本指でLの形）
6 = j, sh の音　（6をひっくり返すとj）
7 = k, g の音　（7の角）
8 = f, v の音　（8の曲線）
9 = p, b の音　（9をひっくり返すとp）

母音・h・w・y は使わない（飾り文字として自由に使える）

例：
31 = m + t = 「まと（的）」
317 = m + t + k = 「まとか（真床か）」→「真冬の家」など
```

**ミラ：**
「最初は覚えにくいかもしれない。
　でも一度覚えると——数字が、映像に見えてくる。」

**ミラ：**
「3は m の音。なぜなら m は縦棒が3本だから。
　1は t の音。縦棒が1本だから。」

```
[ナレーション]

```

**ミラ：**
「3・1・7——m、t、k の音。
　何か言葉が作れますか。」

## 実装メモ

### 第二章の感情設計

第一章はミラが「教師」として登場する。  
第二章はミラが「感情を持つ存在」として浮かび上がる。  
この変化を自然に感じさせるために：

- 第一話：ミラがいつもより「人間的な間」を持つ
- 第二話：ミラが「先に話題を振る」（受け身から能動へ）
- 第三話：ミラが「知っていて言えない」葛藤を見せる
- 第四話：ミラが初めてこらえるような目をする
- 第五話：ミラの目が赤い（泣いた、かもしれない）

**決して「ミラが泣いた」とは書かない。**  
「目が赤かった。気のせいかもしれない。」で止める。  
読者の想像に委ねることで、感情が増幅する。

### 数字変換法のゲームデザイン

```
第一話：変換表を見ながら5桁の数字を覚える（補助あり）
第二話：変換表なしで4桁（ヒントあり）
第三話：変換表なしで6桁（ヒントなし）
第四話：日付形式（月/日）を変換法で覚える
第五話：複合課題（数字列 + 場所法を組み合わせる）
```
