# 記憶の回廊
## 第三章「霧の図書館」— 完全台本

> 記憶術：連想法（Link Method）  
> 全5話 + 隠しエピソード1話  
> 推定プレイ時間：1話あたり10〜15分  
> 前提：第二章「時計塔の秘密」クリア済み

---

## 第三章の構造

**第二章との連続性**
- 「また来てね」と言った声の主に、顔が近づく
- ミラが初めて「あの人」の名前を口にしそうになる
- 館の新しいエリア「霧の図書館」が解放される

**第三章で明かされること**
- 声の主が「女性」であること
- ミラと「あの人」が同じ記憶の中にいたこと
- 僕が記憶を封印した「理由の輪郭」

**第三章で明かされないこと**
- 「あの人」の顔と名前
- 記憶を封印した具体的な出来事
- ミラの本当の正体（第五章まで持ち越し）

## 第一話「霧の中の図書館」

### 冒頭ナレーション

```
十一日目の夢は、白い霧から始まった。

館がある——はずだった。
でも霧が濃くて、輪郭が見えなかった。

歩いていくと、ふいに大きな扉が現れた。
木製の、重そうな扉。
金属の取っ手に、埃が積もっている。

扉の上に、小さなプレートがあった。
「図書館」と書いてある。
```

### ミラとの出会い

```
扉を開けると、ミラがいた。
本棚の間に立って、背表紙を指でなぞっていた。
```

**ミラ：**
「ここは——記憶の図書館です。」

```
[ナレーション]
ミラは振り返らずに言った。
```

**ミラ：**
「あなたが経験したこと、感じたこと、出会った人——
　全部、ここに収められています。」

```
[ナレーション]
見渡すと、天井まで届く本棚が、どこまでも続いていた。
でも——霧がかかっていて、奥が見えなかった。
```

**ミラ：**
「霧が濃いのは、まだ思い出せていない部分が多いから。
　記憶が戻るほど、霧が晴れます。」

### 選択肢A

```
[ナレーション]

```

**ミラ：**
「……あります。」

**ミラ：**
「でも、今はまだ霧の奥です。
　近づくには——もう少し、記憶を戻さなければならない。」

```
[ナレーション]
ミラの声が、一瞬だけ揺れた。
```


## 第二話「連想の糸」

### 冒頭

```
十二日目。

図書館の霧が、昨日より少し薄かった。
本棚の奥が、わずかに見えた。

ノアはカウンターに座っていた。
古い本を読みながら、紅茶を飲んでいた。
```

### 連想法の導入

**ノア：**
「連想法というのはね——」

```
[ナレーション]
ノアは本を閉じた。
```

**ノア：**
「無関係なものを、物語で繋ぐんです。
　『リンゴ』と『電車』と『傘』を覚えたいとする。
　そのまま覚えようとすると、すぐ忘れる。」

**ノア：**
「でも——『リンゴを持った男が電車に乗ったら、
　窓から傘が飛んできた』という話にすると——
　映像になる。映像は、忘れにくい。」

```
[ナレーション]
場所法と似ているが、違う。
場所法は「場所」に置く。
連想法は「物語」で繋ぐ。
```

**ノア：**
「人間の脳は、脈絡のない情報を覚えるのが苦手です。
　でも物語なら——子供でも覚える。
　人類が何千年も物語を語り継いできたのは、そのためです。」

### 選択肢B

> **「記憶と物語は、同じものですか」**

**ノア：**
「鋭い。」

```
[ナレーション]
ノアは眼鏡を押し上げた。
```

**ノア：**
「記憶とは——自分が経験したことを、
　自分なりの物語に編集したものです。
　だから同じ出来事でも、人によって記憶が違う。」

**ノア：**
「あなたの記憶もそうです。
　あの日のことを——あなたはある『物語』として覚えていた。
　でも封印したとき、その物語の糸が切れた。」

```
[ナレーション]
「糸が切れた」——その言葉が、妙にリアルに感じた。
```

---

> **「ミラも物語の一部ですか」**

```
[ナレーション]
ノアはミラを見た。
ミラはノアを見た。
二人の間に、何かが流れた。
```

**ノア：**
「……そうとも言えますな。」

**ミラ：**
「ノア。」

**ノア：**
「言いすぎましたかな。失礼。」

```
[ナレーション]
ノアは苦笑した。
ミラは目を伏せた。
その短いやり取りに、たくさんのことが詰まっていた。
```

---

### 共通ルートへ

**ノア：**
「今日の練習は——この本棚の本を、物語で繋ぐことです。」

```
[ナレーション]
ノアは本棚から、いくつかの本を取り出した。
それぞれの背表紙に、単語が書いてある。
```

**ノア：**
「順番通りに、一つの物語にしてください。
　荒唐無稽でいい。むしろ荒唐無稽な方がいい。
　脳は、普通の話よりも奇妙な話を覚えているものです。」

**ミラ：**
「場所法と組み合わせることもできます。
　図書館の本棚を『場所』にして、
　物語を各棚に置いていく。」

```
[ナレーション]
二つの記憶術が、繋がった気がした。
```

### 第二話エンディング

```
連想の練習をしていると、
ふと——記憶の中に似たような場面が浮かんだ。

誰かと一緒に、何かを覚えようとしていた。
「これとこれを繋げたら、こんな話になる」と笑いながら。

誰と——？

霧の奥に、その人がいる。
```

### 解放される記憶の断片
> *誰かと一緒に、何かを覚えようとした記憶。笑い声が聞こえた。女性の声だった。*

### 次回予告
> 「明日——ノアは、封印された棚に近づく方法を教えてくれた。」

## 第三話「封印された棚」

### 冒頭

```
十三日目。

図書館の霧が、また少し薄くなった。
棚の奥に、他と少し違う棚が見えた。

鎖がかかっている。
```

### ノアとの対話

**ノア：**
「気づきましたな、あの棚に。」

```
[ナレーション]
ノアはカウンターから立ち上がった。
```

**ノア：**
「あの棚には——あなたが一番強く封印した記憶があります。
　鎖は、あなた自身がかけた。」

```
[ナレーション]
近づこうとすると、霧が濃くなった。
体が、自然に足を止めた。
```

**ノア：**
「無理に近づかなくていい。
　今日は——鎖の形を、覚えてもらいます。」

**ノア：**
「鍵は、あなたの記憶の中にある。
　正確に思い出せれば——鎖は外れます。」

### 選択肢C

> **「あの棚に何が入っているか、知っていますか」**

**ノア：**
「知っています。」

```
[ナレーション]
ノアはためらわずに答えた。
```

**ノア：**
「司書ですから。全部、読んでいます。」

**ノア：**
「ただ——読んだことと、あなたに話すことは、別です。
　あなたの記憶は、あなたのものです。
　私が代わりに語ることはできない。」

```
[ナレーション]
その言い方が——公正だと思った。
```

---

> **「なぜ僕はそれを封印したんですか」**

**ノア：**
「……それを知りたいですか。」

```
[ナレーション]
「知りたい」と言いかけた。
でも——本当に知りたいのか、急にわからなくなった。
```

**ノア：**
「知りたくなったときに、棚は開きます。
　知りたくない間は——開かなくていい。」

**ノア：**
「記憶を取り戻すことは、義務ではないですから。」

```
[ナレーション]
その言葉が、不思議と楽にしてくれた。
```

---

### ミラが割り込む

**ミラ：**
「ノア。」

```
[ナレーション]
ミラが、静かに割り込んだ。
```

**ミラ：**
「一つだけ、言っていいですか。」

**ノア：**
「どうぞ。」

**ミラ：**
「あの棚の中の記憶は——悲しいものです。
　でも、恥ずかしいものではない。」

```
[ナレーション]
ミラは封印された棚を見ていた。
```

**ミラ：**
「あなたが封印したのは、弱さからじゃない。
　そのことだけ——先に伝えておきたかった。」

```
[ナレーション]
第一章で、白紙の本に書いてあった言葉を思い出した。
『お前が忘れたのは、弱さではない。』

ミラも——同じことを言っていた。
```

### 第三話エンディング

```
練習を終えると、封印された棚の鎖が、
わずかに緩んでいた。

まだ開かない。
でも——鎖の感触が、少しだけ変わった。

ノアはそれを見て、小さくうなずいた。
```

### 解放される記憶の断片
> *封印した理由は、弱さではなかった。では何だったのか。まだわからない。でも——「悲しいものだ」というノアの言葉が、嘘じゃないことはわかった。*

### 次回予告
> 「明日——ミラが、初めて『あの人』の話を自分からした。」

## 実装メモ

### 第三章のキャラクター設計

**ノアの役割**
- 第一・二章のミラは「感情的サポート」
- ノアは「知的サポート＋時間的距離感」
- 二人が対になることで、物語に奥行きが生まれる
- ノアは「あの人」を知っているが、語らない公正な存在

**ミラの変化（第三章）**
- 第一章：教師
- 第二章：感情を持つ存在
- 第三章：能動的に語る存在（第四話）
- 第四章以降：真実に近づくにつれ、揺れる存在

### 連想法のゲームデザイン

```
第一話：3つの単語を物語で繋ぐ（補助あり）
第二話：5つの単語（補助なし）
第三話：7つの単語 + 順番も覚える
第四話：断片的な「記憶の言葉」を物語にする
第五話：連想法 + 場所法の複合（棚を場所として使う）
```

### 第二章との連続性チェック
- 「また来てね」の声 → 第三章で「女性」と確定
- ミラの「目が赤かった」 → 第三章第四話で能動的に語る伏線として機能
- 第二章の選択肢フラグ → 第三章第四話のミラの語り方に影響
- ノアは第二章の「引き出しの写真」の謎の回収役
