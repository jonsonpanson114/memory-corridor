# 記憶の回廊
## 第四章「廃墟の庭園」— 完全台本

> 記憶術：ストーリー法（Story Method）  
> 全5話 + 隠しエピソード1話  
> 推定プレイ時間：1話あたり10〜15分  
> 前提：第三章「霧の図書館」クリア済み

---

## 第四章の構造

**第三章との連続性**
- 「あの人」の目が見えた——顔が、ついに明かされる
- ミラが「揺れ始める」回
- 館の新しいエリア「廃墟の庭園」が解放される

**第四章で明かされること**
- 「あの人」の顔と、名前の一部（フルネームは第五章）
- 庭に埋められた「もの」の正体
- 僕が記憶を封印した出来事の「直前」まで

**第四章で明かされないこと**
- 封印した出来事そのもの（第五章）
- ミラの正体の詳細（第五章）
- 「もの」を埋めた理由（終章）

## 実装メモ

### 第四章の感情設計

第三章まではミラが「こらえる」存在だった。
第四章でミラが初めて「崩れる」。
ただし崩れ方は静かに——号泣させない。
「背中が震えていた」「目が赤かった」で止める。

**感情の解像度を上げるポイント**
- 第三話：ミラが背を向ける（初めて）
- 第四話：ミラが「好きでした」と言う
- 第五話：ミラが名前を先に言ってしまう（感情が先走る）

この三段階で、ミラが「制御を失っていく」様子を描く。

### ストーリー法のゲームデザイン

```
第一話：4つの断片を時系列に並べる
第二話：6つの断片を因果関係で繋ぐ
第三話：手紙の文章から欠けた単語を補う
第四話：感情のラベルを断片に貼っていく
第五話：全断片を一つの物語として再構成（集大成）
```

### 「まどか」という名前について

第二章でユーザーが「まどか」という言葉を発したとき（選択肢Bで数字変換として）、実はすでに名前が伏線として埋まっていた。
第四章第五話でミラがその名前を言う——この構造が、プレイヤーに「ずっとそこにあった」という感覚を与える。
